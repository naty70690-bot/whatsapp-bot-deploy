# 🚀 DEPLOY DO WHATSAPP BOT - GUIA VISUAL

## 📋 CHECKLIST - Faça nesta ordem:

### ✅ **1. Criar conta GitHub** (se não tiver)
- Acesse: https://github.com
- Cadastre-se gratuitamente
- **Anota seu username!**

### ✅ **2. Criar repositório**
1. Clique: **"New repository"**
2. Nome: `whatsapp-bot-deploy`
3. **Público** (não privado)
4. **NÃO marque** "Add README"
5. Clique: **"Create repository"**

### ✅ **3. Configurar username**
```bash
# Execute este arquivo:
configurar-github.bat
```

### ✅ **4. Deploy automático**
```bash
# Execute este arquivo:
deploy-automatico.bat
```

### ✅ **5. Deploy no Railway**
1. Acesse: https://railway.app
2. **Login** com GitHub
3. **"Start a new project"**
4. **"Deploy from GitHub repo"**
5. Selecione: `SEU_USERNAME/whatsapp-bot-deploy`
6. **"Deploy"**

## 🆘 **PROBLEMAS?**

**Repositório não encontrado:**
- Verifique se criou exatamente `whatsapp-bot-deploy`
- Confirme se é **PÚBLICO**

**Erro de autenticação:**
- Faça login no GitHub no navegador
- Autorize o Railway

**Deploy falhou:**
- Verifique logs no Railway dashboard
- Pode ser erro de build (raro)
