## 🚀 Como Começar

### 1️⃣ Instale Node.js
   - Visite: https://nodejs.org/
   - Baixe a versão LTS
   - ⚠️ Marque "Add to PATH" durante install
   - Reinicie o terminal

### 2️⃣ Abra 2 Terminais

**Terminal 1 - Backend:**
```
cd C:\Users\cauaaa\Documents\whatsapp-bot\backend
npm install
npm start
```

**Terminal 2 - Frontend:**
```
cd C:\Users\cauaaa\Documents\whatsapp-bot\frontend
npm install
npm start
```

### 3️⃣ Acesse no Navegador
```
http://localhost:3000
```

### 4️⃣ Conectar WhatsApp
- Escaneie o QR Code com WhatsApp
- Vá em: Configurações > Aparelhos conectados > Conectar aparelho

### 5️⃣ Configure Respostas Automáticas
- Aba: "Mensagens Automáticas"
- Adicione: Chat ID, Palavra-chave, Resposta
- ✓ Pronto! Bot responderá automaticamente

## 🔗 API Endpoints

### QR Code
- `GET /api/qr`                      → Status + QR
- `GET /api/status`                  → Status conexão

### Mensagens Automáticas
- `GET /api/automated`               → Todas
- `GET /api/automated/:chatId`       → Por chat
- `POST /api/automated`              → Criar
- `PUT /api/automated/:id`           → Editar
- `DELETE /api/automated/:id`        → Deletar

### Clientes
- `GET /api/client`                  → Todos
- `GET /api/client/:chatId/history`  → Histórico
- `POST /api/client`                 → Criar/Atualizar

## 🐛 Problemas Comuns

| Problema | Solução |
|----------|---------|
| npm not found | Node.js não instalado ou terminal não reiniciado |
| Porta 5000 em uso | Mudar `PORT` em backend/.env |
| WhatsApp não conecta | Verificar internet, reiniciar backend |
| BD corrompido | Deletar `whatsapp_bot.db` e reiniciar |
