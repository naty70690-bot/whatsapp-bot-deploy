🚀 COMO USAR (PASSO A PASSO)

   1️⃣  CRIAR BOT NO TELEGRAM
        • Abra Telegram → Procure @BotFather
        • Envie /newbot → Siga instruções
        • COPIE O TOKEN fornecido

   2️⃣  CONFIGURAR NO PAINEL
        • Abra http://localhost:3000
        • Vá para aba "Telegram Bots"
        • Cole o token → "Adicionar Bot"

   3️⃣  CONFIGURAR RESPOSTAS
        • Aba "Mensagens Automáticas"
        • Use "*" para QUALQUER mensagem
        • Funciona para WhatsApp E Telegram!

   4️⃣  PROFIT! 🚀
        • WhatsApp responde automaticamente
        • Telegram responde automaticamente
        • Uma configuração serve para ambos!
