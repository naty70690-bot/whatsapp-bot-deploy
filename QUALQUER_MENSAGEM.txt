================== COMO USAR ==================

1️⃣  Abra o painel em http://localhost:3000

2️⃣  Vá para a aba "Mensagens Automáticas"

3️⃣  Preencha:
   
   Chat ID:       5511999999999  (seu número ou do cliente)
   Palavra-chave: *              ← USE UM ASTERISCO!
   Resposta:      Olá! Obrigado por sua mensagem.
                  Responderemos em breve!

4️⃣  Clique em "Adicionar Mensagem"

5️⃣✅ PRONTO! O bot responderá QUALQUER mensagem para esse contato


================== EXEMPLOS ==================

ANTES (com palavra-chave específica):
└─ Pessoa manda: "olá" → Bot responde ✓
└─ Pessoa manda: "oi" → Bot não responde ✗

AGORA (com asterisco `*`):
└─ Pessoa manda: QUALQUER COISA → Bot responde ✓
   - "olá" → Responde
   - "oi" → Responde  
   - "qual é o preço?" → Responde
   - "😊" → Responde
   - Qualquer mensagem → Responde!


================== MÚLTIPLOS CONTATOS ==================

Se tem vários contatos/números, configure CADA UM:

Contato 1 (5511999999999):
├─ Palavra-chave: *
└─ Resposta: Obrigado! Responderemos em breve.

Contato 2 (5521987654321):
├─ Palavra-chave: oi
└─ Resposta: Oi! Tudo bem?

Contato 2 (5521987654321):
├─ Palavra-chave: *
└─ Resposta: Mensagem recebida!


================== COMBINAR ESTRATÉGIAS ==================

Você pode ter AMBAS para um mesmo contato:

Chat 5511999999999:
├─ "preço" → Resposta específica: "Custa R$ 99,90"
├─ "oi"    → Resposta específica: "Oi! Bem-vindo!"
└─ "*"     → Resposta genérica: "Msg recebida!"

Funcionamento:
• Se disser "preço" → Resposta específica (mais relevante)
• Se disser "oi" → Resposta específica (mais relevante)
• Se disser qualquer outra coisa → Resposta genérica
